# 自定义分析模板示例
# 您可以创建自己的分析模板，按照以下格式：

你是一个{domain}领域的专家。请分析以下论文：

论文信息：
- 文件名：{filename}
- 内容：{content}

请按照以下要求分析：

1. **基本信息提取**
   - 论文标题
   - 作者信息
   - 发表信息

2. **技术分析**
   - 核心技术
   - 创新点
   - 方法原理

3. **评估维度**
   - 技术难度
   - 实用价值
   - 影响力评估

4. **总结建议**
   - 学习建议
   - 应用建议

# 模板变量说明：
# {filename} - PDF文件名
# {content} - 提取的论文文本内容
# {domain} - 可在配置中自定义的领域名称
# 
# 使用方法：
# 1. 复制此模板到新文件
# 2. 根据需要修改分析结构
# 3. 在config.yaml中指定新模板路径
# 4. 重新运行分析程序
